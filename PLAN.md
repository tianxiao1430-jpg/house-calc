# 项目计划书: 在日华人 AI 房产顾问 (House Calc)

> 设计文档: `~/.gstack/projects/house-calc/t_xiao-main-design-20260328-170023.md`

## 项目概述

面向在日华人的手机应用。用户选择「租房」或「买房」模式后，截图房源网站的物件信息，AI 自动识别并通过对话式追问收集个人情况，最终生成**这套房子的完整月度费用明细**。不是买 vs 租的对比工具，而是帮已做决定的用户算清楚具体成本。

**一句话定位：** 替代"群里那个懂日本房产的朋友"。

## 目标用户

- 在日本生活的中文使用者（工作签证、永住、留学等）
- 已经决定租房或买房，需要算清楚一套具体物件的真实月度成本
- 对日本房产费用结构不熟悉，需要中文解释

## 核心功能 (MVP)

1. **模式选择** — 用户选择「租房」或「买房」，进入对应的计算流程
2. **截图识别** — 用户拍照/选取房源截图，AI 根据模式提取对应信息
3. **对话追问** — AI 用中文追问模式相关的个人信息（买房：头金/贷款年限；租房：敷金/礼金等）
4. **费用明细** — 生成该物件的完整月度支出明细 + 初期费用 + 长期累计
5. **历史记录** — 本地保存最近 20 次计算结果

## 技术架构

```
┌─────────────────────────────────────────────────┐
│            Expo App (React Native)               │
├──────────┬──────────┬──────────┬────────────────┤
│模式选择│截图選取│ 対話 UI │費用明細│ 履歴     │
├──────────┴──────────┴──────────┴────────────────┤
│           本地状态管理 (AsyncStorage)              │
└────────────────────┬────────────────────────────┘
                     │ HTTPS
┌────────────────────┴────────────────────────────┐
│           Python 后端 (FastAPI)                   │
├──────────┬──────────────────────────────────────┤
│ API 路由  │  /extract  截图识别                    │
│          │  /chat     对话追问                    │
│          │  /calculate 费用计算 (可选，或前端算)    │
├──────────┼──────────────────────────────────────┤
│ 中间层    │  Rate limit + API key 管理            │
├──────────┼──────────────────────────────────────┤
│ LLM 调用  │  Anthropic Python SDK (Vision+Chat)   │
└──────────┴──────────────────────────────────────┘
```

**关键决策：**
- 费用计算用确定性 TypeScript 代码，不依赖 LLM（准确性 + 成本控制）
- LLM 只负责：截图识别 + 自然语言对话
- **Python (FastAPI) 后端服务**（API key 安全存储、rate limit、后续可扩展用户系统）
- 利率默认 0.475% 变动金利，需防护 0% 边界情况
- LLM 输出通过 Zod schema 严格校验，防止格式/单位错误

## 费用模型详细 (核心 IP)

### 买房侧月度支出

```
月供 = 元利均等返済公式(借入額, 月利率, 月数)
  - 年利率默认 0.475%，月利率 = 0.475% / 12
  - ⚠️ 边界防护: 利率=0% 时特殊处理 → M = P / n
  - ⚠️ 利率验证: 0-5% 范围外报警
管理費 = 截图提取值
修繕積立金 = 截图提取值 (固定)
固定資産税/月 = 分建物+土地计算:
  - 建物: 评估额(市場価×0.70) × 1.4% / 12
  - 土地: 评估额(市場価×0.70) × 1/6(小規模住宅用地特例) × 1.4% / 12
  - ※新築3-5年建物部分再×1/2 (新築減額措置，可选)
都市計画税/月 = 同上拆分，税率 0.3%，土地部分 ×1/3
火災保険/月 = RC造: 面積×200円/12, 木造: 面積×350円/12 (概算)
团信 = 含在利率中 (变动金利通常包含)
```

### 买房初期费用

```
仲介手数料 = 物件価格 × 3% + 6万 + 消費税10%
登録免許税(所有権移転) = 评估额 × 0.3% (中古住宅减免)
  ※新築の場合: 所有権保存登記 = 评估额 × 0.15%
登録免許税(抵当権設定) = 借入額 × 0.1% (住宅减免)
不動産取得税 = max(0, 建物评估额 - 1200万) × 3% + 土地部分別途計算
  ※1200万控除适用于新築住宅，中古有条件适用
印紙税 = 1万~6万 (按价格段)
司法書士報酬 = 10万~15万 (固定概算)
ローン事务手数料 = 借入額 × 2.2%
```

### 租房侧月度支出

```
家賃 = 用户输入
管理費/共益費 = 通常含在物件信息中，默认0
火災保険/月 = 约1.5万/年 / 12 = ¥1,250
更新料摊月 = 家賃 × 1 / 24 (2年1次)
保証会社/月 = 10,000円/年 / 12 = ¥833 (年次更新为固定1万円)
```

### 租房初期费用

```
敷金 = 家賃 × 1~2ヶ月 (默认1)
礼金 = 家賃 × 0~1ヶ月 (默认1)
仲介手数料 = 家賃 × 1.1 (含税)
火災保険 = 约1.5万/年
保証会社初回 = 家賃 × 50%
鍵交換 = 約1.5万~2万
```

### ⚠️ 费用模型注意事项
- 所有金额标注"概算・参考値"，非精确值
- 建议用户以不動産会社的見積書为准
- 固定資産税の実額は市区町村の通知書で確認

## 风险 & 对策

| 风险 | 概率 | 影响 | 对策 |
|------|------|------|------|
| AI 截图识别准确率不够 | 中 | 高 | Fallback 手动输入；优化 prompt；限定支持的平台 |
| API 成本超预期 | 低 | 中 | 费用计算走本地代码；Vision 结果缓存；限制免费使用次数 |
| App Store 审核被拒 | 中 | 中 | 提前研究审核指南；确保隐私政策完整 |
| 费用模型计算不准 | 低 | 高 | 参考实际中介估算验证；标注"仅供参考" |
| 用户不愿回答追问 | 低 | 中 | 提供合理默认值；最多5轮就出结果 |
